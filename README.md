# Library Management System

A multi-platform Library Management System featuring a modern web interface and the original C++ console application.

## 🚀 Web Application (Primary)

The new full-stack version with a **Python (Flask)** backend and a **Premium HTML/CSS/JS** frontend.

### How to Run (Web)
To start the web application easily, use the unified `run.py` script:
```bash
python run.py
```
This will:
1. Start the backend Flask server.
2. Open the frontend dashboard in your default browser.

---

## 🛠️ C++ Version (Legacy)

The original console-based version is available in the `cpp_version/` directory.

### How to Run (C++)
If you are on Windows, you can run the pre-compiled executable:
```bash
./cpp_version/libraryApp.exe
```

---

## Structure
- `backend/`: Python Flask API and data storage.
- `frontend/`: HTML, CSS, and JavaScript files for the dashboard.
- `cpp_version/`: Original C++ source files and executable.
- `run.py`: Unified script to launch the full-stack web app.
